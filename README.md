# Walletboard-debug
Debug firmware for custom board based on Atmega32U4

## Parts
 - 1.3" LED display (SH1106)
 - 6 push buttons
 - 1 WS2812b RGB LED

## Dependencies
 - U8glib
 - Adafruit NeoPixel

## How to build
 1. copy all folders from `./libraries` to your arduino library folder
 2. open `walletboard-debug.ino`
 3. compile it for Arduino Leonardo
